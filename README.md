# Дипломный Проект Movies explorer - бэкенд

### Обзор
*Интерактивный сайт для просмотра, сохранения в избранные и удаления различных фильмов зарегестрированным пользователем.
*Бэкенд подключение к удаленному серверу.

**Методология**
* Express
* Node.js
* MongoDB
* Postman
